Python Bootstrap
=========

Bootstrapper for Python development environment.

Usage
=====

```
    cd bootstrap
    chmod +x bootstrap.sh
    ./bootstrap.sh
```

Debugging
==========

```
    bash -x bootstrap.sh
```
